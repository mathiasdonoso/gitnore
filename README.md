## Getting Started

```
npm install -g gitnore
```

## Usage

To add ignored files for node:
```
gitnore node
```

If you also want to add ignored files for visual studio code:
```
gitnore node --add=visualstudiocode
```
or multiple configurations:
```
gitignore node --add=visualstudiocode,macos,jetbrains,sublimetext
```
